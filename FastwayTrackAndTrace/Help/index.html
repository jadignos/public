Main Form

Below is the user interface of the DAIPOST TrackAndTrace application.

It consists of the following controls.

  • Country - site of the courier
  • Label(s) - input field for tracking numbers or labels. One per line.
  • Status - displays information of the processed labels
  • Track Label(s) - a button to start processing of labels

Sample screen when selecting Country dropdown menu

Sample screen with Label(s) being populated. One line per label.

Sample screen when processing starts by clicking Track Label(s) button. There is a progress bar at the bottom right of the screen to provide status of completion of current process.

Sample screen of the completed process. You can find the generated file in the path provided at the end of the Status window. This is configurable in the Configuration Form.

Configuration Form

This is where you configure the application. You can access this form by clicking File -> Configuration

The Configuration form will pop-up.

In this form, you can configure your settings including where the generated Excel(.xls) file will be saved. Just click the Save button to save your changes. Otherwise, click Close button to close the current window.

NOTE: Make sure to input the correct information, otherwise, the application will have an error when running.

A pop up will be displayed after saving the config file.
